- [x] 参数传递的不同方式（如值传递、引用传递和常量引用传递）

- [ ] 递归函数
- [ ] 常量函数
- [ ] 内存分配和释放函数 new 和 delete
- [ ] 异常处理结构： try、catch 和 throw
- [ ] 类与模板类
- [ ] 类的公有成员、保护成员和私有成员
- [ ] 友元
- [ ] 操作符重载
- [ ] 标准模板库
